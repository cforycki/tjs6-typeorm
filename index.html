Se connecter

Utiliser la méthode 

createConnection(options?: ConnectionOptions) : Promise<Connection>

Plusieurs configurations possibles:

  • ConnectionOptions
  • ormconfig.json
  • Variables d'environnement

ConnectionOptions


export interface ConnectionOptions {
    readonly driver: {
        readonly type: "mysql"|"postgres"|"mariadb"|
                       "sqlite"|"oracle"|"mssql"|"websql";
        readonly url?: string;
        readonly host?: string;
        readonly port?: number;
        readonly username?: string;
        readonly password?: string;
        readonly database?: string;
        readonly extra?: any;
        readonly tablesPrefix?: string;
        [...]
    };
    readonly name?: string;
    readonly entities?: Function[]|string[];
    readonly subscribers?: Function[]|string[];
    readonly dropSchemaOnConnection?: boolean;
    readonly autoSchemaSync?: boolean;
    readonly environment?: string;
    [...]
}

ormconfig.json

Tableau de ConnectionOptions au format JSON

Exemple:


[
  {
    "name": "default",
    "driver": {
      "type": "postgres",
      "host": "localhost",
      "port": 4000,
      "username": "todolist",
      "password": "todolist",
      "database": "todolist"
    },
    "entities": [
      "dist/entities/*.js"
    ],
    "migrations": [
      "dist/migrations/**/*.js"
    ],
    "autoSchemaSync": true
  }
]

Décorateurs

Tables

  • @Entity
  • @AbstractEntity
  • @EmbeddableEntity
  • ...

Colonnes

  • @Column
  • @PrimaryColumn
  • @PrimaryGeneratedColumn
  • @CreateDateColumn
  • @UpdateDateColumn
  • @VersionDateColumn
  • @JoinColumn
  • ...

Possibilité de passer des options (nullité, unicité, valeur par défaut, ...)

Relations

  • @OneToOne (1:1)
  • @ManyToOne (n:1)
  • @OneToMany (1:n)
  • @ManyToMany (n:n)

Possibilité de passer des options (Insertion / Suppression / Modification en cascade)

Exemple


@Entity("my-custom-entity", {'label': 'DESC'})
public class MyEntity {
    @PrimaryGeneratedColumn()
    id: number;
    @Column()
    label: string;
    @OneToMany(
        type => AnotherEntity,
        childObject => childObject.parent,
        { cascadeInsert: true, cascadeUpdate: true }
    )
    childs: Array<AnotherEntity>;
    /* ... */
}

Requêtes

EntityManager

Accessible depuis une connection (connection.entityManager)

  • hasId(entity)
  • create(object)
  • persist(entity)
  • remove(entity)
  • find(Entity)
  • findAndCount(Entity)
  • ...

Repository

Semblable à l'entityManager, mais pour un type d'entité défini

  • hasId(entity)
  • create(object?)
  • persist(entity)
  • remove(entity)
  • find()
  • findAndCount()
  • ...

QueryBuilder

Permet de créer des requêtes complexes
